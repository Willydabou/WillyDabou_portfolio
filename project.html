Memory Game

The Memory Card Game is a browser-based JavaScript game that challenges players to find matching pairs of cards by flipping them two at a time. The goal is to match all 12 pairs (24 cards total) with as few attempts as possible. This game tests the player's memory and focus while providing a fun, interactive experience with immediate feedback and a win notification sound.
